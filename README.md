Just in case if you guys want to proceed with working with github, you need to learn how to push your own updated code to your own branch in here with the command prompt

1. To locate your project folder first:   cd C:\xampp\htdocs\Website (Building)
2. Add your own code first:      git add .
3. Write your commit with this:     git commit -m "Write what you did here"
4. Then push your code to your own branch and then you're done:    git push -u origin YongXun or YeapChearn

   Pull the latest changes of the code your friend did with:       git pull
